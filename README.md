## 🎯 Diferenciais Técnicos e Arquitetura

O grande foco deste portfólio foi aplicar conceitos rigorosos de engenharia de software em uma aplicação comercial de página única:

*   **Métricas e Rastreamento Comercial:** Integração profissional com **Google Tag Manager (GTM)** e **Google Analytics**, permitindo o monitoramento avançado do comportamento dos usuários e cliques em CTAs de conversão.
*   **Performance Otimizada:** Construído sobre o ecossistema do **Vite + React**, garantindo tempos mínimos de carregamento e pontuações de performance de alto nível no Lighthouse.
*   **Experiência de Usuário Fluida (UX/UI):** Uso estratégico do **Tailwind CSS** para design responsivo fluido e microinterações elegantes controladas via **Framer Motion**.
*   **Conformidade com a LGPD:** Implementação de um ecossistema nativo de controle de privacidade com um **CookieBanner** funcional para consentimento do usuário antes do carregamento de scripts de terceiros.
*   **Otimização Avançada de SEO:** Estruturação semântica do HTML para garantir indexação orgânica precisa e visibilidade nos motores de busca (Google).

Um dos módulos principais da aplicação é o sistema interativo de gerenciamento de portfólio, que inclui:
*   **Menu/Carrossel de Seleção:** Permite alterar o projeto em destaque dinamicamente, atualizando textos explicativos, tecnologias aplicadas e lições aprendidas.
*   **Redirecionamento Multicloud:** Link direto para os códigos-fonte no GitHub (para avaliação técnica de código) e botões de acesso rápido às aplicações rodando em produção distribuídas estrategicamente (**3 na Vercel, 1 na Netlify e 1 no GitHub Pages**).

## 🛠️ Stack Tecnológica

| Tecnologia | Finalidade no Projeto |
| :--- | :--- |
| **React** | Biblioteca core para componentização modular, gerenciamento de estados do carrossel e renderização eficiente. |
| **Vite** | Ferramenta de build moderna que substitui o CRA, proporcionando HMR ultrarrápido e bundling otimizado. |
| **Tailwind CSS** | Framework utilitário utilitário para estilização performática e consistência visual responsiva. |
| **Framer Motion** | Engine de animação baseada em física para transições suaves de componentes e interações de scroll. |
| **Google Tag Manager** | Centralização e injeção assíncrona de tags de métricas e analytics. |
